JQurey 개요

경량 javascript 라이브러리
복잡한 javascript 코드를 손쉽게 구현하기 위해 개발됨

JQuery의 장점

1. DOM과 관련된 처리를 쉽게 구현할 수 있음
2. ajax 통신, 이벤트 처리 등을 폭 넓게 지원함
3. 별도의 플로그인을 통해 차트, 슬라이드쇼, 테이블 등을 간단히 구현 가능

JQuery 연결

## head 요소 안에 script 태그를 이용해서 JQuery 파일을 포함시켜줌
<script src='jquery 파일 경로'><script>
1. jquery 홈페이지에서 .js파일을 다운로드 받아 연결하는 방법
2. CDN(Content Delivery Network)을 통한 연결

##script 태그 내에서 jquery 문법으로 코드 작성

(document).ready()

페이지를 모두 로드한 후 ready 메소드를 실행 자바스크립트의 document.onload와 같은 개념의 메소드
